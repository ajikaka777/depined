## Installation

### Prerequisites

Ensure you have the following installed:
- **Python** 
- **pip**

## Installation

1. Clone this repository

```bash
git clone https://github.com/ajikaka777/depined.git
cd depined
```
2. Install dependencies

```bash
pip install -r requirements.txt
```
3. Fill Data and Proxy (Optional)

```bash
nano data.txt
nano proxy.txt
```
4. Run Bot

```bash
python main.py
```
